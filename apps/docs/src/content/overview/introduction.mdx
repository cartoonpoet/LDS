<h2 id="why-lds-matters">LDS가 필요한 이유</h2>

LDS는 법무 서비스의 UI 기준을 한곳에 모아 둔 시스템입니다.  
디자인과 코드 규칙을 함께 관리해서, 화면을 더 빠르게 만들고 일관성도 유지할 수 있게 돕습니다.  
v3는 React 18 기반 서비스를 위한 최신 버전입니다.

<Callout>
  이 문서는 사용하는 사람을 위한 안내서입니다. 컴포넌트 개발과 시각 확인은 Storybook에서 진행합니다.
</Callout>

<h2 id="principles">운영 원칙</h2>

- 먼저 토큰과 기본 컴포넌트를 정리합니다
- 하드코딩 대신 테마 가능한 토큰을 사용합니다
- 문서와 구현 코드는 같은 저장소에서 함께 관리합니다

<h2 id="current-v3-surface">현재 v3 구성</h2>

현재 v3에는 토큰과 기본 컴포넌트가 포함되어 있습니다.  
지금 바로 사용할 수 있는 컴포넌트는 버튼, 인풋, 셀렉트입니다.
