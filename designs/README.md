# Design Drop Zone

Put design references here so I can mirror them in code.

## Recommended

- `assets/`: SVG icons, logos, illustrations
- `components/`: one folder per component, with state screenshots
- `pages/`: full-screen captures
- `references/figma-export/`: exported frames from Figma
- `references/zeplin-export/`: exported screens/spec captures from Zeplin
- `notes/`: copied specs, measurements, interaction notes

## Good file names

- `components/button/primary-default.png`
- `components/button/primary-hover.png`
- `components/input/textfield-focus.png`
- `notes/tokens.md`
- `notes/button-spec.md`
